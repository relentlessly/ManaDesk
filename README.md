# ManaDesk
ManaDesk

This project is a fork of Magic Assistant: https://sourceforge.net/projects/mtgbrowser/
only made possible by the hard work of its creator : Alena Laskavaia
Thank you

ManaDesk allows you to manage your Magic the Gathering cards using a desktop application.
Using Scryfall database, this will allow us to log all your cards, tokens, etc.

One unique feature for traders. You can indexes boxed cards without the need to sort them.
This will allow you to find out valuable cards or wanted cards without losing the time to sort them first.
